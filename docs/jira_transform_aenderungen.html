Neue Funktion: Get-EpicStatusPivot  Neu
Nach Abschnitt 6 (Get-SprintName) einfügen
▶

Für Epics: letzter Eintrittszeitpunkt je Status als Dictionary. Im Gegensatz zur Dual-Period-Logik für Stories zählt hier nur der neueste Wert pro Status.

function Get-EpicStatusPivot {
    param($Issue)
    $pivot = @{}
    if (-not ($Issue.PSObject.Properties['changelog'] -and
              $Issue.changelog.PSObject.Properties['histories'])) {
        return $pivot
    }
    foreach ($h in ($Issue.changelog.histories | Sort-Object { [datetime]$_.created })) {
        foreach ($item in $h.items) {
            if ($item.field -eq "status") {
                $s = $item.toString
                if (-not $pivot.ContainsKey($s) -or
                    [datetime]$h.created -gt [datetime]$pivot[$s]) {
                    $pivot[$s] = $h.created   # letzter Wert gewinnt
                }
            }
        }
    }
    return $pivot
}

Neue Funktion: Convert-EpicToRow  Neu
Nach Get-EpicStatusPivot einfügen
▶

Baut eine Epic-Zeile mit Summary, Stage, EPIC-Kategorie und dynamischen Status-Spalten (aus Get-EpicStatusPivot). Epics erhalten keine Resolved/Rejected/Sprint-Felder.

function Convert-EpicToRow {
    param($Issue, [string]$SquadName, $CustomFields)
    $f = $Issue.fields

    $stageRaw = if ($f.PSObject.Properties[$CustomFields.epicStage]) {
                    $f.($CustomFields.epicStage) } else { $null }
    $stage    = if ($stageRaw -and $stageRaw.PSObject.Properties['value']) {
                    $stageRaw.value } else { "$stageRaw" }

    $catRaw   = if ($f.PSObject.Properties[$CustomFields.epicCategory]) {
                    $f.($CustomFields.epicCategory) } else { $null }
    $cat      = if ($catRaw -and $catRaw.PSObject.Properties['value']) {
                    $catRaw.value } else { "$catRaw" }

    $pivot = Get-EpicStatusPivot -Issue $Issue

    $row = [ordered]@{
        "Jira-ID"              = $Issue.key
        "Kurzbeschreibung"     = $f.summary
        "Issue-Status"         = $f.status.name
        "Stage"                = $stage
        "EPIC-Kategorie"       = $cat
        "Created (Status New)" = $f.created
        "Squad"                = $SquadName
        "Project-Key"          = $f.project.key
    }
    foreach ($key in ($pivot.Keys | Sort-Object)) { $row[$key] = $pivot[$key] }
    return $row
}

Neue Funktion: Get-BlockedEpisodes  Neu
Nach Convert-EpicToRow einfügen
▶

Liest aus dem Changelog jede Blocked-Episode: Start = toString gesetzt, Ende = fromString gesetzt. Paart Start- und End-Events der Reihe nach (BlockedSeq). Berechnet Dauer in Stunden und „seit wie vielen Tagen blockiert".

function Get-BlockedEpisodes {
    param($Issue, [string]$SquadName, $CustomFields)
    $episodes = [System.Collections.Generic.List[object]]::new()
    if (-not $Issue.changelog.PSObject.Properties['histories']) { return $episodes }

    $starts = [System.Collections.Generic.List[string]]::new()
    $ends   = [System.Collections.Generic.List[PSCustomObject]]::new()

    foreach ($h in ($Issue.changelog.histories | Sort-Object { [datetime]$_.created })) {
        foreach ($item in $h.items) {
            # Feld per ID oder Anzeigename erkennen
            if ($item.field -eq $CustomFields.blockedReason -or
                $item.field -eq "Blocked Reason" -or
                $item.field -eq "Blockiert/Wartend") {
                if (-not [string]::IsNullOrWhiteSpace($item.toString))   { $starts.Add($h.created) }
                if (-not [string]::IsNullOrWhiteSpace($item.fromString)) {
                    $ends.Add([PSCustomObject]@{ Time = $h.created; Reason = $item.fromString })
                }
            }
        }
    }

    for ($i = 0; $i -lt $starts.Count; $i++) {
        $start = $starts[$i]
        $end   = if ($i -lt $ends.Count) { $ends[$i] } else { $null }
        $durH  = if ($end) { [math]::Round(([datetime]$end.Time - [datetime]$start).TotalHours, 2) } else { $null }
        $seit  = if (-not $end) { ([datetime]::Now - [datetime]$start).Days }
                 elseif ($durH) { [math]::Ceiling($durH / 24) } else { $null }

        $episodes.Add([PSCustomObject]@{
            "Jira-ID"              = $Issue.key
            "Status"               = $Issue.fields.status.name
            "Squad"                = $SquadName
            "BlockedStart"         = $start
            "BlockedEnd"           = if ($end) { $end.Time } else { $null }
            "BlockedReason"        = if ($end) { $end.Reason } else { $null }
            "BlockedSeq"           = $i + 1
            "BlockedDurationHours"  = $durH
            "BlockiertWartendSeit"  = $seit
        })
    }
    return $episodes
}
